DaughterDB can be downloaded from this URL:

http://fields.scripps.edu/DaughterDB/20021228-DaughterDB.zip

The password is "ions" to extract the Zip file.  For Windows machines,
the program can be installed by extracting the Zip to the directory
"C:\DaughterDB" and copying the "DaughterDB.bat" file to the Windows
directory.

To see the list of command-line options for the program, type the
following at a command prompt window:

DaughterDB --help
